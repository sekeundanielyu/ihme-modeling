This directory contains:

- the congenital registry data prep code used for GBD 2016 ("01_registries")
- the congenital lit extraction data prep code used for GBD 2016 ("02_lit_extractions")

- code used to upload the prepped registry and lit data to the Epi Uploader ("03_initial_uploads")
- code used to upload hospital data ("04_hospital_uploads")

- code that squeezes the sum of the specific sub-cause prevalence estimates to DisMod-MR model total prevalence estimates ('squeeze')

- code that adjusts the denominators of Turner and Klinefelter Syndrome data to be sex-specific & applies age-at-diagnosis correction factors to these data
